If you have found an error, please view the Known Bugs before you write one of the guys!

NOTE: Please use the following type of template with all bug reports:

  • Amiga OS verions (kickstart and workbench)
  • Extensions running (MagicWB, NewIcons, DirOPUS, ToolManager, etc.)
  • Settings of Amiga that caused the problem
  • HostOS (Win95, Win95-OSR2, WinNT, etc.)
  • DirectX version
  • PC Graphics card and its driver's version
  • Sound card and its driver's version
