Over the years, WinUAE has proven to be run pretty much all Amiga software ever created.

If you have problems to get a certain program working, look at the links page. There you will find the most comprehensive emulation sites on the net.

The number of programs that don't run has been reported to be close to zero. If you find a piece of software to require special treatment, please let the developers know. Look at the contacts page to get in touch with the right people.

Still, WinUAE is not perfect, but please check this list before reporting any bugs:

  • Some CD32 games stop loading and freeze (bug in the CD-ROM controller emulation, hard to fix because there are no CD32 hardware specs)
  • Sound and JIT-sound timing problems (some systems only)

NOTE: Please use the following type of template with all bug reports:

  • WinUAE Version
  • Host Operating System (Windows XP, Vista, 7, 8 etc.)
  • DirectX version
  • PC Graphics card and the driver version
  • Sound card and the driver version
  • Amiga OS version (Kickstart and Workbench)
  • Extensions running (MagicWB, NewIcons, DirOPUS, ToolManager, etc.)
  • Amiga settings that may have caused the problem

If you made sure your request is still valid, see the Credits page to contact one of the developers.
