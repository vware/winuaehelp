Over the years, WinUAE has proven to be run pretty much all Amiga software ever created.

If you have problems to get a certain program working, look at the links page. There you will find the most comprehensive emulation sites on the net.

The number of programs that don't run has been reported to be close to zero. If you find a piece of software to require special treatment, please let the developers know. Look at the contacts page to get in touch with the right people.
