Priority and Extensions

 

When Active

When the program is active and running you can set priority to different priorities to improve performance of WinUAE than other programs.
Settings include Low, Below Normal, Normal, Above Normal and Highest.

 

When InActive

  • When the program is active and running you can set priority to different priorities to improve performance of other programs when WinUAE is not in use. Settings include Low, Below Normal, Normal, Above Normal and Highest.
  • Disable sound turns off audio output when minimized.
  • Pause emulation will completely stop emulation when WinUAE is minimized.
  • Disable game controllers will completely stop game controller emuation.

 

When Minimized

  • Run at lower priority makes the emulation use less CPU resources when WinUAE is minimized. You can specify the level of priority to use: Highest, Above Normal, Normal, Below Normal, and Low.
  • Disable sound output turns off audio output when minimized.
  • Pause emulation will completely stop emulation when WinUAE is minimized.
  • Disable game controllers will completely stop game controller emuation.

 

File Extension Associations

Here you can associate certain file extensions with the Winuae program so that when you double click file you can auto load Winuae for you. If you have Amiga Forever 2008, it comes with the Amiga Player which allows you to auto-run adf games for you. Either double click an extension to associate Winuae with it or use the Associate all button. Use Deassociate all to remove all associations.
