Disk Swapper

This tab can contain upto 20 disk images. Double clicking on a line will bring up a file requester, where you can select a disk image to add to the list.

Select a disk image from the pull down disk list and click Insert disk image to add it to the Disk image list.

Selecting an image and clicking Remove disk image will remove the image from the list.

Selecting the drive will specify which drive the image will appear in (DF0-DF3). Up to four images in four drives can be specified.

Once the disks have been selected, they will appear in the appropiate Disk Drives slots.
