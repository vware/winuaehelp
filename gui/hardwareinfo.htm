Hardware Info



  A list of hardware attached to the emulator once the configuration is defined. For example, it will list disk interfaces (IDE or SCSI), ROMs used, RAM used, graphics cards, accelerator boards, and others.


Type will display interface type. Z2 = Zorro 2, Z3 = Zorro 3.

Name will display the name of the hardware.

Start will display the starting memory address of the device.

Size will display the size of memory used.

ID will display the ID number of the device used by AutoConfig.

 

Custom board order changes the order of the devices.

Move up will move an intem further up the list.

Move down will move an item further down the list.
