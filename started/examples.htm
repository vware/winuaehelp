All the other settings can be set as you wish, the ammount of RAM can be adjusted up or down to suit the software.
If using Workbench, then I suggest using as much Fast RAM or Z3 RAM as you need - typically, 4-32MB would cover most software applications.
The A500 configuration with a Kickstart 1.3 ROM and 512kB or 1MB Chip RAM would work. For AGA games the A1200 configuration would work, adding some Fast RAM can help a lot.
For PowerPC Emulation you only use Accelerator RAM, plus a Piccasso IV rom, cyperstorm or blizzard ppc roms and the QEmu-uae.dll plugin.
When saving a configuration, enter the name of the Amiga with a few details of its configuration would help to you to select which configuration is needed.
If using one configuration a lot, save it as default and then WinUAE will use this configuration without you having to Load it everytime, just click OK to start!
