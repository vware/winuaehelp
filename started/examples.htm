Introduction

To get you started at emulating Amigas you need to know the configuration of the most popular or frequently used Amigas to get your games or software working.
All the other settings can be set as desired. If using Workbench, it is suggested using as much Fast RAM or Z3: 8-32MB would cover most software applications.
Adding some Fast RAM can help a lot.
For PowerPC Emulation you only use Accelerator RAM, plus a Picasso IV ROM, Cyberstorm or Blizzard PPC ROMs and the qemu-uae.dll plugin.
